# Taskipelago/Taskipelabingo Version 1.0.0
*Finally out of beta!*

## New Features
* Description text can be attached to tasks
* Added a YAML option to preview task rewards via scouting or hinting

## Changes
* Added dollar signs to the purchase button to make it more distinctive

## Bug Fixes
* Fixed a bug where consumable items wouldn't register when placed after filler items

## Known Issues
* Universal Tracker is unhappy with the structure of the datapackage because of the dynamic nature of Taskipelago's location names. This is aimed to be resolved in a future patch. This is an issue specifically with multiple taskipelago worlds in a single multiworld with shared item or location names.
